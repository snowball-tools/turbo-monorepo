# Monorepo

This is just a starter monorepo with api (express) and web (vite + react) using shared packages.

## Install Dependencies (in root)

```bash
pnpm install
```

## Run 

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Lint

```bash
pnpm lint
```
